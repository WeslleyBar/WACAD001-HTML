A LINGUAGEM DE PROGRAMAÇÃO

HyperText Markup Language, Linguagem de Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software). Um software pode ser desenvolvido para rodar em um computador, dispositivo móvel ou em qualquer equipamento que permita sua execução.

Alguns dos recursos da linguagem de Programação vistos até este momento:

Recursos Tags associadas Descrição
Links <a> Criar links entre páginas
Listas <ul> e <li> Criar listas de itens ordenados
Listas <ol> e <li> Criar listas de itens não ordenados
Imagens <img> Inserir imagens
